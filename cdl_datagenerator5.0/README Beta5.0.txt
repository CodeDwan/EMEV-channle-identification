——————3.0版分割线——————
这是数据生成器Beta 3.0版
本次更新主要包含以下内容：
1. 针对CDL-A/B/C/D/E模型设置了不同的DelaySpread，具体设置参考3GPP的TR.38.901。
2. 删除了80km/h和120km/h的高速移动用户，新增了40km/h的移动速度。
3. 确保不同的速度是近乎均匀的分布在数据集中。

这是数据生成器Beta 4.0版
本次更新主要包含以下内容：
1. 新增了保存每个样本对应的用户速度功能，保存数据中加入键值“Vmatrix”

______5.0版分割线————————
这是数据生成5.0版本，该版本是一个重要的更新版本，主要更新如下：
1. 该版本舍弃了OFDM符号数维度，因为该维度所有数据几乎相同
2. 该版本不再使用子载波概念，而是选择1个RB（即12子载波）作为最小分辨率
3. 该版本大幅度提高了RB个数
4. 将downlink进行了转置输出，其维度与uplink同步，即(子载波x基站天线x用户天线)
